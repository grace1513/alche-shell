# Loops, Conditions and Parsing

This project contains Bash scripts that demonstrate the use of loops, condition statements, and text parsing in shell scripting.

## Files

- 1-for_best_school: Displays "Best School" 10 times using a for loop.

## Requirements

- Ubuntu 20.04 LTS
- Bash scripts must be executable
- Must pass ShellCheck without errors
- No use of awk
